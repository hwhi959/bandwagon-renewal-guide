## 搬瓦工续费时间线：从到期到删除，每一步都在

很多人慌的原因是不清楚到底有多少时间窗口。把节奏说清楚：

**到期前 7 天** → 系统提前生成续费账单（搬瓦工近期更新了这个机制，比之前宽裕）

**到期当天** → 账单确认生成，邮件通知发出

**到期后 7 天宽限期** → VPS 完全正常运行，你可以在这段时间随时付款

**宽限期过后** → 服务暂停（Suspend）。注意，暂停阶段账单通常还有效，付了款还能恢复

**长时间不付** → 服务终止（Terminate），数据永久清空，不可恢复

所以实际上你有接近两周的时间窗口处理续费。真正需要避免的是到了 Terminate 那一步——到那时候，就算你愿意付钱也找不回数据了。

## 确认续费开关有没有被关掉

这步很多人忽略了。

后台里有个 Renewal Invoices 的开关，默认是 ON。如果不小心把它设成了 OFF，到期就不会生成账单，系统不通知你，机器直接到期删掉。

检查方法：登录后台 → Services → My Services → 找到对应 VPS → 点小齿轮 → Manage Renewal → 确认状态是 ON。

一分钟的事，但确实有人因为这个丢过机器。

## FAQ

**Q：搬瓦工 $19.99 续费后会自动扣信用卡吗？**

不会。搬瓦工官方明确不会自动从绑定的支付方式扣款。"自动续费"是指账户余额充足时自动扣余额，但余额需要你自己提前充进去。

**Q：快到期了后台没有续费账单，正常吗？**

正常。账单在到期当天（美国时区）或到期前 7 天才生成，到期前几个月找不到续费入口是正常现象，不用担心。

**Q：我把续费周期从年付改成月付，价格会变吗？**

会，而且变动非常大。修改周期后系统按当前官方售价重算，持有老价格套餐的用户基本等于放弃老价格保护，不建议操作。

**Q：到期忘了续费，VPS 被暂停了还能恢复吗？**

暂停阶段付款通常可以恢复，但如果进入了 Terminate（终止）状态，数据永久删除，无法恢复。别等到这一步。
